album.getTags

Description

Get the tags applied by an individual user to an album on Last.fm

Parameters

Class Call

  • api_key (required): A Last.fm API key
  • album (optional): The album name in question
  • artist (optional): The artist name in question
  • mbid (optional): The musicbrainz id for the album

Method Call

  • sessionKey (required): The session key returned from auth.getSession
  • secret (required): The secret for your API account
