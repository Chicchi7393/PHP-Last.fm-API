artist.getInfo

Description

Get a list of upcoming events for this artist

Parameters

Class Call

  • api_key (required): A Last.fm API key
  • artist (required): The name of the artist
  • mbid (optional): The musicbrainz id of the artist

Method Call

None

Example Usage

// Include the API
require '../../lastfmapi/lastfmapi.php';

// Setup the variables
$apiKey = '';
$artist = 'Athlete';

// Create the class
$apiClass = new lastfmApi($apiKey, $artist);

// Call the method
if ( $return = $apiClass->getInfo() ) {
	// $return contain the returned array
}
else {
	// Have a better error handeling system here
	die('Error '.$apiClass->error['code'].' - '.$apiClass->error['desc']);
}
