album.getInfo

Description

Get the metadata for an album on Last.fm using the album name or a musicbrainz id. See playlist.fetch on how to get the album playlist

Parameters

Class Call

  1. api_key (required): A Last.fm API key
  2. album (optional): The album name in question
  3. artist (optional): The artist name in question
  4. mbid (optional): The musicbrainz id for the album

Method Call

None

Example Usage

// Include the API
require '../../lastfmapi/lastfmapi.php';

// Setup the variables
$apiKey = '';
$artist = 'Green day';
$album = 'Dookie';

// Create the class
$apiClass = new lastfmApiAlbum($apiKey, $artist, $album);

// Call the method
if ( $return = $apiClass->getInfo() ) {
	// $return contain the returned array
}
else {
	// Have a better error handeling system here
	die('Error '.$apiClass->error['code'].' - '.$apiClass->error['desc']);
}
